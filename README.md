# In-Context Fine-Tuning

## Development

Use [uv](https://docs.astral.sh/uv/) for package management.

Run `make install` to setup a local envrionment, see [Makefile](./Makefile) for
other development commands.

## Usage

The cli contains scripts for fine-tuning, prompt-tuning and utilities.

```bash
uv run cli.py --help
```
